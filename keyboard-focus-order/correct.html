Correct: There is a logical tab order

Tab through the links on this page and check out that the focus moves logically through the links.

We could change how the tab index works by setting it to a value greater than 0. That would move it ahead of other elements on the page, but "Using a tabindex greater than 0 is considered an anti-pattern." Source: Google's Web Fundamentals Guide about Accessibility
